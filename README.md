## üìÇ Dataset Description

The dataset contains detailed information related to **bank loan applications and repayments**. Key fields include:

* Loan details: `loan_amount`, `term`, `int_rate`, `installment`
* Borrower information: `annual_income`, `emp_length`, `home_ownership`, `address_state`
* Credit & risk metrics: `dti`, `grade`, `sub_grade`, `verification_status`
* Loan performance: `loan_status`, `total_payment`, `Good Loans vs Bad Loans`
* Time-based fields: `issue_date`, `last_payment_date`, `next_payment_date`

## üîç Project Workflow

### 1. Data Loading & Preparation

* Loaded the dataset into Excel
* Reviewed column formats and corrected data types
* Ensured consistency in categorical and date fields
* Prepared clean data for analysis and reporting

---

### 2. Exploratory Data Analysis (EDA)

* Analyzed distribution of loan amounts, interest rates, and annual income
* Examined borrower profiles by employment length and home ownership
* Evaluated loan performance using **loan status** and **good vs bad loan classification**
* Identified trends across time and categories

---

### 3. Pivot Table Analysis

Created pivot tables to analyze:

* Total number of loans issued
* Total funded loan amount
* Total amount received from borrowers
* Good loans vs bad loans comparison
* Loan performance by:

  * State
  * Purpose
  * Grade and sub-grade
  * Term duration

Slicers were added to enable dynamic filtering and easy exploration.

---

### 4. Excel Dashboard Creation

Developed an **interactive Excel dashboard** that includes:

* **Key Performance Indicators (KPIs)**

  * Total Loans Issued
  * Total Funded Amount
  * Total Payment Received
  * Average Interest Rate
  * Average Debt-to-Income (DTI)
* Visualizations for:

  * Loan status breakdown
  * Good vs bad loans
  * Loan distribution by state and purpose
  * Trends over time
* Slicers for real-time filtering by loan attributes

The dashboard is designed for clarity, usability, and executive-level reporting.
