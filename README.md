## Project Overview

This project aims to perform an analysis on **bank loan data** using **Microsoft Excel** to understand the trends in the loan data and differentiate between good and bad loans. The analysis follows a complete data analytics workflow from raw data exploration to an interactive dashboard.

The project will be conducted to demonstrate the skills required to perform data analytics using **Excel data analytics skills** for the banking industry 

---

## Tools & Technologies

* **Microsoft Excel**

  * Pivot Tables & Pivot Charts
  * Excel formulas and aggregations
  * Slicers and filters
  * Dashboard design
* **Data Analysis Techniques**

  * Exploratory Data Analysis (EDA)
  * KPI 
  * Trend and categorical analysis

---

## Dataset Description

The dataset contains information related to **bank loan applications and repayments**. Key fields include:

* Loan details: `loan_amount`, `term`, `int_rate`, `installment`
* Borrower information: `annual_income`, `emp_length`, `home_ownership`, `address_state`
* Credit & risk metrics: `dti`, `grade`, `sub_grade`, `verification_status`
* Loan performance: `loan_status`, `total_payment`, `Good Loans vs Bad Loans`
* Time-based fields: `issue_date`, `last_payment_date`, `next_payment_date`

---

## Project Workflow

### 1. Data Loading & Preparation

* Loaded the dataset into Excel
* Checked the format of all columns and corrected the data type if necessary  
* Ensured data consistency for categorical dates and variables
* Prepared clean data for analysis and reporting

---

### 2. Exploratory Data Analysis (EDA)

* Analyzed distribution of loan amounts, interest rates, and annual income
* Examined borrower profiles by employment length and home ownership
* Evaluated loan performance using **loan status** and **good and bad loan comparison **
* Identified trends across time and categories

---

### 3. Pivot Table Analysis

Created pivot tables to analyze:

* Total number of loans issued
* Total funded loan amount
* Total amount received from borrowers
* Average interest rates
* Average Debt to income (DTI)
* Month to Date
* Month over Month 
* Good loans vs bad loans comparison
* Loan performance by:

  * State
  * Purpose
  * Employment length 
  * Term duration

---

### 4. Excel Dashboard Creation

Created an **interactive Excel dashboard** that includes:

* **Key Performance Indicators (KPIs)**

  * Total Loans Applications
  * Total Funded Amount
  * Total Amount Received
  * Average Interest Rate
  * Average Debt-to-Income (DTI)
* Visualizations for:

  * Loan status breakdown
  * Good vs bad loans
  * Loan distribution by state and purpose
  * Trends over time
* Real time filtering based on laon characteristics

---

## Key Insights

* some grades of loans and purpose of loans have higher risk patterns
* Loan performance varies significantly by state and borrower profile
* Interest rates and DTI play a major role in loans
